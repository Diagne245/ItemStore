  1. About:

    2. This is a productivity App to help store Ideas & Todos. It brings us a way to organize either our ideas or our tasks at hands into editable groups. The App offers an easy way to add, update and classify them. It integrates also a focus area where we can add our motivation quotes.

    The stored items can be viewed on all your devices, Mobile as well as Desktop. We recommand the later for better User Experience though. So let's not let any great idea get lost any more!

  2. How To:
    • Add Item:  Enter the item content and push the add Item Button.
    • Update Item:  Click on any added item to enter Edit Mode. You can then update it by editing its content.
    • Delete Item:  To remove an item, click it and press the "X" icon on its top right corner. A Confirm prompt will appear.
    • Add Group:  To add a new Group, enter the group title and push the add Group Button.
    Each group can compose of multiple slides to further categorize items.
    • Add New Slide:   hit the + icon at the bottom of the slide.
    • Remove Slide:   press the trash can icon on the top right corner and confirm.
    • Delete Group:   remove all its slides using the method exposed above.
    To handle operations on multiple items, we have implemented a Select Mode
    • Enter / Exit Select Mode:   Double Click any item to enable select mode. Double Click again to disable it and return to normal mode.
    • select items:  While in select mode, single click any item to select it.
    • Move selected items:  While in select mode, navigate to any location and hit the Add Selection Button.
    • delete selected items:  While in select mode, click the Remove Items Button to delete your selection.
